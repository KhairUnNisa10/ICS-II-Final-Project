About Us

At the end of our 2nd semester in university, we were given a project in which we had to make a web application using any language. As it was a web application we decided to make the project using HTML, CSS, JavaScript and JSON as these are the basic languages used in everything related to the web. Then we had many ideas on what to make but after long discussions in the group, we decided to make a Quiz Application with a decent look yet attractive UI. A quiz can be defined as a game or brain teaser to test knowledge. It can contain an element of competition when participants play against each other to get the highest score, which helps participants become more engaged.

As HTML is a language used to structure the web app, we used HTML 5 to structure the whole web application and invest a really good amount in perfecting the structure so it doesn't create any problems in the future.

As for the designing part, we used CSS 3 to make the web application as much attractive as we can. Designing also plays an important role as GUI is the first thing a user interacts and feels good by looking at it.

As for the JavaScript, we have used JavaScript Vanilla for all the functionality happening in the web app. Other than HTML & CSS, the main part was of JavaScript which actually made the whole project work smoothly by applying different logics.
